
Kotlin, Spring Boot, MySQL, JPA, Hibernate Rest API
Build a Restful CRUD API using Kotlin, Spring Boot, Mysql, JPA and Hibernate.

Requirements
Java - 11.0.19

Maven - 3.8.8

Mysql - 5.7.42

Spring-Boot - 2.7.0

Endpoints: 

GET /api/articles

POST /api/articles

GET /api/articles/{id}

PUT /api/articles/{id}

DELETE /api/articles/{id}
You can test them using postman or any other rest client.
